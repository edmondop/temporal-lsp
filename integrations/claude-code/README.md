# temporal-lsp

Temporal workflow anti-pattern detection for Claude Code. Flags non-deterministic operations, bad signatures, and workflow pattern violations.

## Supported Extensions
`.py`, `.go`

## Installation

Install the LSP binary:

```bash
go install github.com/edmondop/temporal-lsp/cmd/temporal-lsp@latest
```

Make sure `$GOPATH/bin` is in your PATH.
